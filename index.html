📊 Google Sheets Sync

Conecta con Google Sheets para que tú y tu esposa vean los mismos datos en tiempo real desde cualquier iPhone.

📋 Cómo configurar:
1
Ve a sheets.google.com y crea una hoja nueva llamada "MisPagos"
2
En la hoja, ve a Extensiones → Apps Script
3
Pega el código que aparece abajo y dale clic en Implementar → Nueva implementación → Aplicación web. Copia la URL.
4
Pega la URL aquí abajo y comparte la hoja con el email de tu esposa
function doPost(e){ var lock=LockService.getScriptLock();lock.tryLock(10000); try{var s=SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var d=JSON.parse(e.postData.contents);s.clearContents(); s.appendRow(["id","nombre","monto","estado","vencimiento","categoria","notas","fecha_pago","icono"]); d.forEach(function(r){s.appendRow([r.id,r.name,r.amount||"",r.status,r.due||"",r.category,r.notes||"",r.paid_date||"",r.icon]);}); }finally{lock.releaseLock();} return ContentService.createTextOutput(JSON.stringify({result:"ok"})).setMimeType(ContentService.MimeType.JSON); } function doGet(e){ var s=SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var rows=s.getDataRange().getValues();rows.shift(); var data=rows.map(function(r){return{id:r[0],name:r[1],amount:r[2],status:r[3],due:r[4],category:r[5],notes:r[6],paid_date:r[7],icon:r[8]};}); return ContentService.createTextOutput(JSON.stringify(data)).setMimeType(ContentService.MimeType.JSON); }
🔔 Notificaciones Push

Recibe alertas automáticas en tu iPhone cuando un pago está próximo a vencer.

1
Primero instala la app en tu pantalla de inicio: Safari → Compartir 📤 → "Agregar a pantalla de inicio"
2
Abre la app desde el ícono en tu pantalla de inicio (no desde Safari)
3
Toca el botón de abajo — el iPhone te pedirá permiso para enviar notificaciones
